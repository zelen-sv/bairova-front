# Bairova Portfolio

**Start project**

1. Install dependencies:
```bash
npm i
```
2. Create env file '.env.local' in root directory with text:
```ruby
VUE_APP_API_URL=http://localhost:3000
```
3. To start the development server, run:
```bash
npm run serve
```

App running at:
http://localhost:8080/
